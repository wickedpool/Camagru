<h2>My own INSTAGRAM !</h2>
<br>

<h3>This is the fist projet right after the PHP piscine, We have to create, with php and some JavaScript a litle Instagram website.</h3>

<hr>

<h3>Users can take a webcam shot and then do some "photo-montage", pictures will pop in the public gallery and users can comment, like or delete their post.</h3>

<hr>

<h3>Frameworks are forbidden so we have to do everything. from registration to css style.</h3>

<hr>

<h3>100% responsive, with my own media query's.</h3>

<hr>

<h2>How to use?</h2>

<h3>Clone</h3>

This is a PHP project so you have to start LAMP, MAMP, WAMP or other apache server.
Then you can clone this repo in your apache diretory like htdocs for MAMP.

      git clone https://github.com/wickedpool/Camagru-42.git Camagru
   
Then you must go to <code>localhost:<PORT>/Camagru/install.php</code> to install the database and fill it.

Once your database is created you can create an account and ENJOY.
